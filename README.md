### Installation

1. Clone the repo
```sh
git clone https://github.com/dzenan-ordagic97/eTransportAPI.git
```
2. Install Docker Desktop 
```sh
https://www.docker.com/products/docker-desktop
```
3. Open project folder and run command prompt 
```JS
docker-compose build
```
```JS
docker-compose up
```
